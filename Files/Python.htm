Python

Python - высокоуровневый объекто-ориентированный язык программирования общего назначения, поддерживающий императивное, процедурное, структурное, объектно-ориентированное программирование. Стандартная библиотека включает большой набор полезных переносимых функций, начиная с возможностей для работы с текстом и заканчивая средствами для написания сетевых приложений.

Преимущества:

· Динамическая типизация данных

· Читаемость кода

· Минималистичный синтаксис

Недостатки:

· Невысокая скорось работы

· Высокое потребление памяти
