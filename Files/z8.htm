№ 1.

  Сколько слов длины 4, начинающихся с согласной буквы и заканчивающихся гласной буквой, можно составить из букв М, Е, Т, Р, О? Каждая буква может входить в слово несколько раз.

Решение

Напишем программу для получения ответа.

PascalABC Python
###
'МЕТРО'.Cart(4).Cnt(s-> (s[1] in 'МТР') and (s[^1] in 'ЕО')).Pr 		from itertools import *
print(len([s for s in product('МЕТРО',repeat=4) if s[0] in 'МТР' and s[-1] in 'ЕО']))

Ответ: 150.

№ 2.

  Ольга составляет 5-буквенные коды из букв О, Л, Ь, Г, А. Каждую букву нужно использовать ровно 1 раз, при этом Ь нельзя ставить первым и нельзя ставить после гласной. Сколько различных кодов может составить Ольга?

Решение

Напишем программу для получения ответа.

PascalABC Python
###
'ОЛЬГА'.Prm.Sel(x-> x.ToS(''))
  .Cnt(s-> (s[1]<>'Ь') and |'АЬ','ОЬ'|.All(c-> c not in s)).Pr 		from itertools import *;k=0
for x in permutations('ОЛЬГА'):
  s=''.join(x)
  k+=s[0]!='Ь' and all(c not in s for c in ['АЬ','ОЬ'])
print(k)

Ответ: 48.
