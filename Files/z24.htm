№ 1.

  Текстовый файл состоит не более чем из 106 символов L, D и R. Определите длину самой длинной последовательности, состоящей из символов L. Хотя бы один символ L находится в последовательности.

Задание 24-1

Решение

Напишем программу для получения ответа.

PascalABC Python
###
var (k,m):=(0,0);
ReadAllText('24-1.txt').ForEach(c->
  if c='L' then k+=1
  else begin
    remax(m,k);k:=0;
  end);
Pr(m) 		m=k=0
for c in open('24-1.txt').read():
  if c=='L': k+=1
  else: m=max(m,k);k=0
print(m)

Ответ: 7.

№ 2.

  Текстовый файл содержит строки различной длины. Общий объём файла не превышает 1 Мбайт. Строки содержат только заглавные буквы латинского алфавита (ABC…Z).
  В строках, содержащих менее 25 букв A, нужно определить и вывести максимальное расстояние между одинаковыми буквами в одной строке.

Задание 24-2

Решение

Напишем программу для получения ответа.

PascalABC Python
###
ReadLines('24-2.txt').Wh(s-> s.Cnt('A')<25)
  .Max(s-> s.Max(c-> LastPos(c,s)-Pos(c,s))).pr 		m=[]
for s in [s for s in open('24-2.txt').readlines() if s.count('A')<25]:
  m+=[s.rfind(c)-s.find(c) for c in s]
print(max(m))

Ответ: 1004.
